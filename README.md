# IPL Cricket Analytics Dashboard
Interactive Power BI dashboard analysing 17 seasons of IPL data

## Dashboards
- Dashboard 1: Match Overview & Tournament Summary
- Dashboard 2: Batting & Scoring Analytics
- Dashboard 3: Bowling & Wicket Analytics

## Dataset
- matches.xlsx — 1,095 match records
- deliveries.csv — 2,60,920 ball-by-ball records

## Tools Used
Power BI · DAX · Power Query · Excel
